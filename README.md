# Diesel Sustainability Analysis Across Telecom Network Sites

This repository contains my Data Analytics II case study on diesel sustainability across selected telecom network sites.

## Project Overview

The project analyses anonymised operational spot-check data to understand how diesel level, grid availability, and consumption per day affect available days across selected telecom network sites.

The aim is to support better operational decision-making around diesel planning, site prioritisation, and proactive risk management.

## Business Question

How do diesel level, grid availability, and consumption per day affect available days across telecom network sites?

## Methods Used

The analysis applies the following techniques:

- Exploratory Data Analysis
- Data Visualisation
- Hypothesis Testing
- Correlation Analysis
- Regression Analysis

## Files in this Repository

- `Victoria_Idegwu_Data_Analytics_II_CASE1.qmd` - Quarto source file containing the full analysis code and report.
- `Victoria_Idegwu_Data_Analytics_II_CASE1.html` - Rendered HTML report.
- `DA_Exam_Data_Anonymised.csv` - Anonymised dataset used for the analysis.

## Confidentiality Note

Actual site identifiers were anonymised using generic labels such as `Site_001`, `Site_002`, and `Site_003`.

This anonymisation only affects site names and does not change the numerical data used for the statistical tests, visualisations, correlation analysis, or regression model.
